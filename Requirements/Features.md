Features:

- Unified Interactable Academic Calendar: Displays classes, assignments, exams, and events in a single calendar so that students can have a simple interface that gives them a bird’s eye view of everything they have going on. The user will be able to edit dates of tasks and activities if needed.  

- Assignment and Exam Tracking: Allows students to add and manage academic deadlines, giving reminders so that they don’t slip from the student’s mind.  

- Event Manager: Tracks club meetings and extracurricular activities, separate from academic trackers so that the student can balance between academics and social life when needed. The user will be able to see how their various commitments might overlap for efficient planning. 

- Task Prioritization: Sorts tasks by how much of the final grade they are worth and how much a student values the specific class. This allows students to focus more time on assignments and assessments that are more important without having to think too much about them. 

- Highest Priority: The Unified Interactable Academic Calendar is the highest priority feature because it directly addresses the main goal of the system: helping students stay organized and is necessary for the other features to build off.
